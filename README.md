商品管理系统2.0 <br>(Spring + SpringMVC + MyBatis + MySQL)
===
     
    
     

### 功能介绍:<br>
* 对商品信息的查,增,改,删;<br>
* 添加商品时对输入的信息校验;<br>
* 增加了登录页面;
* 查询的结果在分页中显示;<br>

### 运用知识:<br>
* SSM框架的使用与整合;<br>
* WEB三层架构设计思想及SpringMVC运行流程;<br>
* 注解及XML文件配置依赖关系;<br>
* MyBatis的自动代理生成DAO的实现类;<br>
* 在服务器端使用Validator对post请求参数校验;<br>
* 使用SpringMVC的Intercepter拦截请求实现登录功能;<br>
